Entendendo O Git & Github.

  1. Git diff
    2. Mostra a diferença no arquivo que não foram realizados.
  2. Git add . arquivos.
    3. Faz o snapshop de um arquivo na preparação para versionamento.
  3. Git diff-staged
    4. Mostra a diferença entre arquivos selecionados a suas últimas versões.
  4. Git commit -m "Menssagem descrita"
    5. Grava o snapshop permanentemente do arquivo no historico de versão.
  5. Git status
    6. Lista todos os arquivos novos ou modificados para serem commitados.
  6. Git branch
    7. Lista todos as branches locais no repositorio atual.
  7. Git Branch "Main"
    8. Cria uma nova (branch ) Muda para o branch especifico e atualiza o diretorio de trabalho.
